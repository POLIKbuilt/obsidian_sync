| Пьян                   |                                            | 6-10 пив, 5 стопок водки  |
| Сильно Пьян            |                                            | 11-15 пив, 7 стопок водки |
| Алкогольное Отравление |                                            | 20 пив, бутылка водки     |
| Пьяный Обморок         |                                            | Все что дальше            |
